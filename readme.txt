== Description ==

SWG Theme is a feature-rich WordPress theme specifically designed for Star Wars Galaxies gaming communities. It provides extensive customization options, security features, performance optimization, and seamless integration with popular WordPress plugins.

= Key Features =

* **Advanced Admin Customization**
  - Custom WordPress admin bar with scrollable menu
  - Clean Admin mode to hide theme pages from Appearance menu
  - Comprehensive Theme Info page with system diagnostics
  - Tabbed system information (Host, PHP, Apache, MySQL)

* **Security Features**
  - Login security with failed attempt limiting
  - IP blocking and whitelist management
  - File integrity monitoring
  - Database security hardening
  - Custom admin URLs
  - Security headers configuration

* **Performance Optimization**
  - Asset minification and combination
  - Image lazy loading
  - Browser caching
  - Database query optimization
  - CDN integration support

* **User & Social Features**
  - Enhanced author profiles
  - Social share counts
  - Social proof popups
  - Related authors
  - Author follow system
  - User ratings

* **SEO & Analytics**
  - Advanced SEO settings
  - Schema markup
  - Google Analytics integration
  - XML sitemap generation
  - Meta tag optimization

* **Plugin Compatibility**
  - WooCommerce (E-commerce)
  - Yoast SEO
  - Elementor
  - Jetpack
  - Advanced Custom Fields (ACF)
  - WPML (Multi-language)
  - BuddyPress
  - The Events Calendar
  - Easy Digital Downloads
  - Contact Form 7
  - Gravity Forms
  - bbPress
  - SWG Auth Plugin

* **Content Management**
  - Built-in image gallery system
  - Multiple sidebar areas
  - Custom post types support
  - Widget-ready areas
  - Customizable menus

* **Design Features**
  - Dark mode support
  - Responsive design
  - Customizable colors
  - Typography options
  - Search functionality

== Installation ==

1. Upload the theme files to `/wp-content/themes/swgtheme/` directory
2. Activate the theme through the 'Appearance > Themes' screen in WordPress
3. Configure theme options via the WordPress admin bar menu or Appearance > Theme Options
4. Set up your preferred settings in each tab

== Frequently Asked Questions ==

= How do I access theme settings? =

Theme settings can be accessed through:
- WordPress Admin Bar > SWG Theme menu (when logged in)
- WordPress Admin > Appearance > Theme Options

= What is Clean Admin mode? =

Clean Admin mode hides all theme-related pages from the Appearance menu, providing a cleaner WordPress admin interface. Pages remain accessible via direct URLs and the admin bar menu.

= How do I view system information? =

Go to WordPress Admin Bar > SWG Theme > Theme Info to view comprehensive system diagnostics including PHP, Apache, MySQL, and host system information.

= Is the theme compatible with page builders? =

Yes, the theme is compatible with Elementor and other popular page builders.

= Does the theme support WooCommerce? =

Yes, full WooCommerce support is included with custom styling options.
